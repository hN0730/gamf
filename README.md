# 🎵 ACC Music Festa 2025 Website

광주 ACC 뮤직페스타 공식 웹페이지입니다.  
**HTML**, **CSS**, **JavaScript**로 제작되었으며,  
픽셀아트 감성의 노을 테마를 중심으로 디자인되었습니다.

---

## 📁 프로젝트 구조

📂 website

├── html/ (페이지별 HTML 파일)

├── style/ (CSS 스타일 파일)

├── js/ (기능 스크립트)

├── images/ (이미지 및 아이콘)

├── font/ (사용 폰트)

└── index.html (메인 페이지)


---

## 💡 주요 기능
- 상단 메뉴 및 서브 메뉴 hover 효과  
- 자동 카운트다운 타이머  
- 반응형 디자인 (Tailwind + Custom CSS)  
- 페이지별 독립 스타일 적용
