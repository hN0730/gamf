# 🎵 ACC Music Festa 2025 Website

광주 ACC 뮤직페스타 공식 웹페이지입니다.  
**HTML**, **CSS**, **JavaScript** 로 제작되었으며,  
픽셀아트 감성의 노을 테마를 중심으로 디자인되었습니다.

---

## 📁 프로젝트 구조

📂 website

├── style/ (CSS 스타일 파일)

├── js/ (기능 스크립트)

├── images/ (이미지 및 아이콘)

├── font/ (사용 폰트)

└── index.html (메인 페이지)

└── notice.html (축제 공지 페이지)

└── festivalmap.html (행사장 장소 안내 페이지)

└── lineup.html (아티스트 라인업 페이지)

└── timetable.html (시간별 공연 안내 페이지)

└── ticket.html (티켓 안내 페이지)

└── earlyticket.html (얼리티켓 안내 페이지)

└── gallery.html (사진 페이지)

---

## 💡 주요 기능
- 상단 메뉴 및 서브 메뉴 hover 효과  
- 자동 카운트다운 타이머  
- 페이지별 독립 스타일 적용
